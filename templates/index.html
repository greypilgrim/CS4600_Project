Secure Encryption App

Welcome to the Secure Encryption Demo

This application demonstrates how modern cryptography protects sensitive information using a hybrid approach with RSA and AES encryption.

What You'll Learn

  • Asymmetric Encryption: How RSA public/private key pairs allow secure key exchange
  • Symmetric Encryption: How AES provides efficient encryption of message content
  • Message Integrity: How HMAC ensures messages haven't been tampered with
  • Hybrid Systems: How combining multiple cryptographic techniques creates robust security

Technical Overview: This demo implements RSA-2048 for key exchange, AES-256-CBC for message encryption, and HMAC-SHA256 for message authentication.

Key Generation

Generate RSA public/private key pairs for secure asymmetric encryption.

Encryption

Encrypt messages using AES with the key protected by RSA encryption.

Decryption

Decrypt messages by first recovering the AES key, then decrypting the content.
